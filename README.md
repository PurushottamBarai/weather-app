# Weather Application

## Overview

A modern and clean weather application built with HTML, CSS, and JavaScript. It provides real-time weather information for any city using the OpenWeatherMap API. The project features a responsive glassmorphism design with a dark theme.

## Features

- Real-time weather data fetching.
- Search functionality by city name.
- Displays city, country, temperature, and weather conditions.
- Modern glassmorphism user interface.
- Smooth animations and transitions.

## Technologies Used

- HTML5
- CSS3 (Vanilla CSS, Flexbox, Animations)
- JavaScript (Fetch API, Async/Await)
- OpenWeatherMap API
